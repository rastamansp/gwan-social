# ADRs — Architecture Decision Records

Coloque aqui decisões arquiteturais numeradas cronologicamente.

## Convenção de nomes

- Formato: `NNNN-titulo-curto-em-kebab-case.md`  
- Exemplo: `0001-uso-de-redis-como-fila-e-cache.md`

## Conteúdo

Use o template em [../00-governance/adr-template.md](../00-governance/adr-template.md).

## Numeração

- `0001`, `0002`, … sem saltos no mesmo repositório (facilita auditoria).
